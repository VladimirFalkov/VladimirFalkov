# Vladimir Falkov

**Full-stack / backend developer — I build and ship complete business applications, front to back.**

Django & FastAPI backends *and* React / TypeScript & HTMX frontends, for systems that connect to real
hardware: cameras and ANPR, field sensors, GNSS/RTK and access control. Solo, end to end, in production —
one person for the whole product.

## Selected work

- **[access-control-anprv](https://github.com/VladimirFalkov/access-control-anprv)** — ANPR-based vehicle access & parking control for a gated community. Live cameras, licence-plate recognition, RBAC, real-time guard monitor, PWA. *FastAPI · React · Hikvision · solo · in production.*
- **[estateos-property-platform](https://github.com/VladimirFalkov/estateos-property-platform)** — multi-tenant property access & operations platform: ANPR access, tenant directory, service requests and billing in one system. *Django · ~40 sites · in production.*
- **[soil-penetrometer-platform](https://github.com/VladimirFalkov/soil-penetrometer-platform)** — multi-tenant agritech platform for soil-penetrometer data: geolocated readings (PostGIS), optimal GPX sampling routes, depth analysis. *Django · HTMX · PostGIS.*
- **[rtk-ntrip-caster](https://github.com/VladimirFalkov/rtk-ntrip-caster)** — multi-tenant NTRIP caster delivering RTK GNSS corrections from base stations to field rovers, with a live network map and per-tenant access. *C caster core · Django · in production.*
